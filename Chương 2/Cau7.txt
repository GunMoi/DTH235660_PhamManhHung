```plaintext
Một số cách nhập dữ liệu từ bàn phím trong Python:

1. Sử dụng hàm `input()` để nhập dữ liệu dạng chuỗi:
   ```python
   name = input("Nhập tên: ")
   ```

2. Nhập và chuyển đổi sang kiểu số nguyên:
   ```python
   age = int(input("Nhập tuổi: "))
   ```

3. Nhập và chuyển đổi sang kiểu số thực:
   ```python
   score = float(input("Nhập điểm: "))
   ```

4. Nhập nhiều giá trị trên một dòng, dùng `split()`:
   ```python
   a, b = input("Nhập hai số, cách nhau bởi dấu cách: ").split()
   a = int(a)
   b = int(b)
   ```
